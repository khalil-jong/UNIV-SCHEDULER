DOSSIER lib/ - Dépendances externes
===================================

Placer ici le fichier mysql-connector-j-8.0.33.jar

TÉLÉCHARGER:
1. Aller sur: https://dev.mysql.com/downloads/connector/j/
2. Télécharger: mysql-connector-j-8.0.33.jar (ou version plus récente)
3. Copier le fichier JAR dans ce dossier (lib/)

Le fichier JAR sera automatiquement reconnu par Eclipse grâce au .classpath
